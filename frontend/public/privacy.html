Privacy Policy

Last updated: March 15, 2025

Overview

ScoreGenius is a Progressive Web App that provides real-time NBA analytics, predictive insights, and betting recommendations. We are committed to protecting your privacy and being transparent about our data practices.

Limited Use

  • All data is used solely for providing NBA analytics, game predictions, and betting insights.
  • We do not sell, rent, trade, or transfer any user data to third parties.
  • Data collection is limited to only what is necessary for the app's core functionality.
  • No advertising or marketing use of any collected data.

Use of Permissions

Our application requires minimal permissions to function:

  • Storage Permission: Used only to save your preferences and cached game data locally
  • Host Permission (sportsgenius.io): Required only for secure data retrieval and API access

Data Collection

Our application collects and stores the following information locally on your device:

  • User preferences (favorite teams, dark mode settings, notification preferences)
  • Game prediction history for performance tracking
  • Temporary game data cache for offline functionality

Note: No personal information is collected or stored beyond what you explicitly provide.
